# 背光控制应用

这是一个Android应用，用于控制设备的背光，需要ROOT权限。

## 功能特点

- 申请ROOT权限
- 自动搜索背光文件路径
- 手动设置背光文件路径
- 快速关屏功能
- 简单易用的界面

## 最低系统要求

- Android 7.0 (API 24) 或更高版本
- ROOT权限

## 在AIDE中构建项目

1. 下载AIDE应用（可以从Google Play商店或其他应用商店下载）
2. 打开AIDE应用
3. 选择"导入项目"
4. 导航到项目目录 `BacklightControlApp`
5. 等待AIDE加载项目
6. 点击"构建"按钮
7. 构建完成后，点击"运行"按钮安装应用

## 可能遇到的问题及解决方法

1. **找不到gradle-wrapper.jar**
   - AIDE会自动处理Gradle依赖，无需手动下载

2. **构建失败**
   - 确保AIDE已更新到最新版本
   - 确保设备已连接到互联网，以便AIDE下载必要的依赖

3. **应用崩溃**
   - 确保设备已获取ROOT权限
   - 检查背光文件路径是否正确
   - 尝试在设置中手动指定背光文件路径

4. **背光文件路径**
   - 常见的背光文件路径：
     - `/sys/class/backlight/*/brightness`
     - `/sys/devices/soc/*/leds/lcd-backlight/brightness`
     - `/sys/class/leds/lcd-backlight/brightness`

## 如何使用

1. 打开应用
2. 授予ROOT权限
3. 点击主界面的圆形按钮关闭屏幕
4. 点击右上角的设置按钮进入设置界面
5. 在设置界面中可以手动指定背光文件路径和启用快速关屏功能

## 注意事项

- 本应用需要ROOT权限才能工作
- 不同设备的背光文件路径可能不同
- 请谨慎使用，不当操作可能会损坏设备
